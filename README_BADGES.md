[![PRs Welcome](https://img.shields.io/badge/PRs-welcome-brightgreen.svg)](http://makeapullrequest.com)

## Badge Explanations

1. **CI/CD Pipeline**: Shows if tests are passing
2. **Python Version**: Indicates minimum Python version
3. **Code Style**: Shows we use Black formatter
4. **License**: Project license type
5. **PRs Welcome**: Encourages contributions
